# LogiSearch - Motor de Búsqueda Logística con IA

## Descripción
Buscador de rutas logísticas con análisis de costes, comparador de transportistas y generación de RFQs.

## Arquitectura
- **Frontend**: React 19 + TypeScript + MUI 7 + Framer Motion 12
- **AI**: Gemini AI + Tavily web search
- **DB**: Supabase (PostgreSQL)
- **Export**: jsPDF para RFQs

## Características
- ✅ Búsqueda en lenguaje natural
- ✅ Análisis de rutas y costes
- ✅ Comparador de transportistas
- ✅ Generación de RFQs con PDF
- ✅ Historial de búsquedas
- ✅ Calculadora de costes interactiva
- ✅ PWA con code splitting

## Mejoras Fusionadas
- ✅ Multi-provider AI con key rotation
- ✅ Calculadora de costes con tarifas reales
- ✅ Export profesional de RFQs
- ✅ Búsqueda semántica con Tavily
- ✅ Supabase para persistencia
