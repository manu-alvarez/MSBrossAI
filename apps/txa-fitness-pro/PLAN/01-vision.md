# Visión del Proyecto

## TxaFitnessPro - Behavioral Change Modeling Platform

### Propósito
Plataforma de Modelado del Cambio Conductual que integra un motor de diagnóstico
inteligente (Diagnostic Funnel) con un sistema de coaching proactivo basado en
adherencia continua.

### Objetivos Clave
1. **Diagnóstico Preciso**: Evaluación multi-dimensional mediante scoring ponderado
   con detección de correlaciones entre dimensiones.
2. **Planes Personalizados**: Generación dinámica de módulos de coaching basados
   en el perfil de debilidades detectado.
3. **Adherencia Medible**: Sistema de tracking de actividades con métricas objetivas
   y retroalimentación continua.
4. **Arquitectura Escalable**: Diseño híbrido CAMS (Content-Aware Microservices)
   preparado para MLOps y personalización por IA generativa.

### Stack Tecnológico
- **Frontend**: Next.js 15 (App Router) + React 19 + Tailwind CSS v4
- **State**: Zustand + persistencia local
- **Backend**: Supabase (PostgreSQL + Auth + RLS)
- **Engine**: TypeScript puro con algoritmo de scoring multidimensional
- **Despliegue**: Vercel Edge Functions

### Roadmap
- **Sprint 0**: Setup y fundamentos (completado)
- **Fase 1**: Diagnóstico Inteligente y Coaching Base
- **Fase 2**: Adhesión, Gamificación y Bucle de Feedback
- **Fase 3**: Personalización Profunda e IA Generativa
